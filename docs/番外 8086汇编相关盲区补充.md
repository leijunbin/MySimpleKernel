# 8086汇编相关补充

## 关于org

`nasm`汇编编译器为`bin`文件格式提供了额外的操作符`org`，它的功能是指定程序被载入内存时的起始地址

## int 0x10中断

当`ae、ax`或`eax`存在不同值时，执行不同的功能，主要与屏幕相关

+ `ax：3`: 清除屏幕全部字符

+ `ah: 0x0e, al: 字符`: 打印对应字符

## 寄存器相关

+ `ah`：`ax`高八位寄存器

+ `al`：`ax`低八位寄存器

+ `ax`：`eax`低十六位寄存器

+ `eax`：三十二位寄存器

## jmp跳转代码段地址

```s
jmp 0:0x1002
```
